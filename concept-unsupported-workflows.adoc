= 지원되지 않는 워크플로
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
다음 워크플로는 VM 1.0 릴리즈에 대한 BlueXP 백업 및 복구에서 지원되지 않습니다.

* VMDK 연결 및 분리
* 데이터 저장소를 마운트하고 마운트 해제합니다
* 게스트 파일 및 폴더를 복구합니다
* 대체 위치로 복원합니다
* "인증서" 인증 방법을 사용하여 스토리지 시스템을 추가합니다
* NetApp ONTAP FlexGroup 볼륨을 추가합니다
